# Auto-Trader UI Specification

This document provides detailed specifications for the Auto-Trader monitoring and control interface.

---

## Overview

The Auto-Trader UI provides real-time monitoring and control of automated trading sessions. It supports both paper trading (safe testing) and live trading (with MT5 integration on Windows).

### LivePositionCard

Displays a single open position with real-time P&L.

```typescript
interface LivePositionCardProps {
  position: {
    symbol: string;
    direction: 'long' | 'short';
    size: number;
    entryPrice: number;
    currentPrice: number;
    stopLoss: number;
    takeProfit: number;
    pnl: number;
    pnlPercent: number;
  };
  onClose?: () => void;
}
```

### SignalLog

Streaming log of trading signals and decisions.

```typescript
interface SignalLogProps {
  signals: Signal[];
  maxEntries?: number;
  streaming?: boolean;
}

interface Signal {
  timestamp: Date;
  symbol: string;
  action: 'BUY' | 'SELL' | 'HOLD' | 'CLOSE';
  prediction: number;
  confidence: number;
  executed: boolean;
  reason?: string;
}
```

### SessionEquityChart

Real-time equity curve with streaming updates.

```typescript
interface SessionEquityChartProps {
  data: { timestamp: Date; equity: number }[];
  streaming?: boolean;
  height?: number;
}
```

### TradeNotification

Toast notification for trade events.

```typescript
interface TradeNotificationProps {
  type: 'opened' | 'closed' | 'sl_hit' | 'tp_hit' | 'error';
  symbol: string;
  direction?: 'long' | 'short';
  pnl?: number;
  message?: string;
}
```
